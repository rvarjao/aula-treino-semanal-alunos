# Modelo do Aluno — Programa Semanal de Treino

## Objetivo
Completar um sistema simples para entender a integração entre:
- backend
- banco de dados
- frontend

## Arquivos
- `app.py`
- `schema.sql`
- `static/style.css`

## O que falta fazer
Você deverá completar os pontos marcados com `TODO`:

- TODO 1: criar o banco usando o arquivo `schema.sql`
- TODO 2: fazer a consulta `SELECT`
- TODO 3: mostrar os treinos na tela
- TODO 4: fazer o `INSERT`
- TODO 5: revisar o formulário
- TODO 6: completar a tabela no `schema.sql`
- TODO 7: melhorar o CSS

## Como rodar
1. Instale o Flask:
   `pip install flask`

2. Execute:
   `python app.py`

3. Abra:
   `http://localhost:5000`

## Dica
Faça um TODO por vez e teste o sistema a cada mudança.
